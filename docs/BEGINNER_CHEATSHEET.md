## 中文

**你只需要判断 3 件事**：① 能用起来吗 ② 看得懂吗 ③ 文字/排版正常吗（**有没有乱码**）。

### 方式 A — Slack（最简单，已装好）
在 Slack 工作区 **Biomate** 里，任意频道或私聊 BioMate 机器人，输入下面任意一条：

1. `/biomate help` → 应弹出帮助
2. `/biomate Screen aspirin CC(=O)Oc1ccccc1C(=O)O for hERG and CYP3A4`
   → 几秒内“正在分析”，随后给出结果 **+ 一个 Run in BioMate 按钮**
3. `/biomate 查询 UniProt P04637` → 应返回 p53 蛋白信息

### 方式 B — 网页版
浏览器打开 **https://test.stage-public.biomate.ai**，用发给你的账号登录，
直接用中文提问，例如：`筛选阿司匹林的 hERG 和 CYP3A4 安全性`。

### 通过 = 几秒内有**清晰、不乱码**的回复，且有结果/按钮可点。

**遇到问题怎么报告**：截图 + 写下你**输入了什么** + 大概**时间**，发给我们。

## English

**Judge just 3 things:** ① does it work ② can you understand it ③ is the text /
formatting clean (**any garbled characters?**).

### Option A — Slack (easiest, already installed)
In the **Biomate** Slack workspace, in any channel or a DM to the BioMate bot,
type any of:

1. `/biomate help` → a help message should pop up
2. `/biomate Screen aspirin CC(=O)Oc1ccccc1C(=O)O for hERG and CYP3A4`
   → "analyzing…" within 3 s, then a result **+ a Run in BioMate button**
3. `/biomate Look up UniProt P04637` → should return p53 protein info

### Option B — Web app
Open **https://test.stage-public.biomate.ai**, log in with the account you were
given, and just ask in plain language, e.g. `Screen aspirin for hERG and CYP3A4`.

### Pass = a clear, non-garbled reply within seconds, with a result / clickable button.

**To report an issue:** screenshot + **what you typed** + roughly **when**, send it to us.
